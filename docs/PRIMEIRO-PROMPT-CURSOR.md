# Primeiro prompt para o Cursor

Use este prompt depois que o repositorio estiver aberto no Cursor.

```text
Leia README.md, docs/CURSOR-CONTEXTO.md, docs/ARQUITETURA.md e docs/fases/00-BOOTSTRAP.md.

Execute apenas a fase 00-BOOTSTRAP.

Nao rediscuta arquitetura, stack, produto ou escopo. Corrija somente o que for necessario para validar o bootstrap do projeto.

Ao final, responda exatamente neste formato:

## Resultado

Resumo curto do que foi implementado ou validado.

## Arquivos alterados

- caminho/arquivo

## Como testar

1. passo objetivo
2. passo objetivo

## Testes executados

- comando: resultado

## Pendencias

- pendencia objetiva ou "Nenhuma pendencia conhecida"
```
